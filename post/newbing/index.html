NewBing（新必应）中国大陆使用教程（含手机端）

2023-02-23 2 min read # NewBing # 教程

一、NewBing 内测申请教程

1.使用现有微软账号登录Microsoft Learn Lab获得临时UK虚拟服务器。

2.在虚拟服务器上申请新Microsoft账号。

3.在虚拟服务器上使用新Microsoft账号访问 NewBing 加入等待申请内测并等待通过。

二、NewBing PC使用教程

1.本地下载Edge edv版。

2.安装 Header Editor 浏览器拓展。

3.添加Header Editor 拓展规则：修改全部请求头，头名称为”x-forwarded-for”、头内容为非中国大陆区IP。

4.登录内测申请通过的Microsoft账号即可访问NewBing。

三、NewBing 安卓和鸿蒙系统手机使用教程

1.下载 Yandex 浏览器。（其他可安装浏览器拓展的手机浏览器亦可）

2.下载改浏览器UA的插件User-Agent Switcher。
（为了修改UA，NewBing 仅限微软自家UA使用）
修改UA的拓展

3.使用User-Agent Switcher添加浏览器UA：
Mozilla/5.0 (Linux; Android 12; 22041216UC) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/112.0.0.0 Mobile Safari/537.36 EdgA/112.0.0.0

（为了让Bing识别Yandex浏览器为Edge Dev版）
添加新UA：Edge Dev版

4.安装 Header Editor 浏览器拓展。

5.添加Header Editor 拓展规则：修改全部请求头，头名称为“x-forwarded-for”、头内容为非中国大陆区IP。
修改请求头

6.登录内测申请通过的Microsoft账号即可使用手机访问NewBing。
拿来学习效率很高

注：无法下载Yandex浏览器和插件的话可以使用Kiwi浏览器替代（该浏览器可以添加本地.crx拓展），Kiwi浏览器及拓展下载-123云盘。
